# Install with precompiled binaries

<Tabs groupId="operating-systems">

<TabItem value="windows" label="Windows">

1. Download the precompiled binary for your system from [GitHub Releases](https://github.com/vertex-center/vertex/releases/latest).

2. Run vertex
    ```Bash
    .\vertex.exe
    ```

3. Open the UI in your browser: [http://localhost:6130](http://localhost:6130).

</TabItem>

<TabItem value="macos" label="macOS">

1. Download the precompiled binary for your system from [GitHub Releases](https://github.com/vertex-center/vertex/releases/latest).

:::info
Download the <code>arm64</code> version if you have a mac M1 or newer.
:::

2. Run vertex
    ```Bash
    ./vertex
    ```

3. Open the UI in your browser: [http://localhost:6130](http://localhost:6130)

</TabItem>

<TabItem value="linux" label="Linux">

1. Download the precompiled binary for your system from [GitHub Releases](https://github.com/vertex-center/vertex/releases/latest).

2. Run vertex
    ```Bash
    ./vertex
    ```

3. Open the UI in your browser: [http://localhost:6130](http://localhost:6130)

</TabItem>

</Tabs>
